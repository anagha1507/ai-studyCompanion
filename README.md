# AI Study Companion

A smart study companion application built with Python and Streamlit.

## Features
- AI-powered study assistance
- Voice interaction with Google Text-to-Speech
- Document analysis with ChromaDB
- Machine learning capabilities with PyTorch and scikit-learn

## Tech Stack
- **Frontend**: Streamlit
- **Backend**: FastAPI
- **AI/ML**: PyTorch, Transformers, scikit-learn
- **Database**: ChromaDB
- **Voice**: gTTS, OpenAI Whisper
